📖 Guide d'utilisation

🖱️ Navigation de base

Clic simple sur une tuile → ouvre le lien dans un nouvel onglet.
Clic long / Clic droit sur une tuile → affiche le menu d'actions (éditer, dupliquer, supprimer).
Glisser-déposer → réorganisez vos tuiles en les faisant glisser d'une position à une autre.

➕ Ajouter une tuile

Cliquez sur une case vide (pointillés) pour y ajouter un lien ou un dossier.
Renseignez l'URL et le nom, puis choisissez une image via :

  • AUTO — récupère automatiquement le favicon du site.
  • GOOGLE — recherche une image sur Google Images.
  • Upload depuis votre appareil ou collez un lien direct.

📂 Dossiers

Cliquez sur une case vide et choisissez "Dossier" pour créer un groupe de liens.
Un clic sur la tuile dossier ouvre une fenêtre flottante avec sa propre grille de liens.
Glissez des tuiles depuis la grille principale vers un dossier (et inversement) pour les déplacer.

🔍 Barre de recherche

Trois moteurs disponibles en bas de page : Google, Google Images et DuckDuckGo.
Tapez votre recherche et appuyez sur Entrée pour ouvrir les résultats dans un nouvel onglet.

⚙️ Paramètres (menu latéral)

Ouvrez le menu via le bouton "Paramètres" en bas à gauche pour :

  • Changer la couleur de fond et la couleur des tuiles.
  • Ajuster le nombre de colonnes, de lignes et l'espacement.
  • Modifier la taille et la police du texte.
  • 📤 Exporter votre configuration en fichier JSON (sauvegarde).
  • 📥 Importer une configuration JSON précédemment sauvegardée.
  • 📑 Importer des favoris depuis un fichier HTML exporté de votre navigateur.
  • ↩ Annuler — revient à la dernière sauvegarde automatique.

💡 Conseils & astuces

  • Vos données sont sauvegardées automatiquement dans le navigateur (localStorage).
  • Pensez à exporter régulièrement votre config pour éviter toute perte.
  • Les images encodées en base64 sont stockées localement — pas besoin d'internet.
